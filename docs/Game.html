Namespace: Game

Game

Main simulation values, where the Ships are stored, and the main loop.
Properties:
Name Type Description
gravitationalAcceleration number How much velocity to apply each iteration.
exhaustSpread number Modifies how much random spread there is on exhaust.
exhaustRandomness number Modifies how much particles move randomly each iteration.
minExhaustVelocity number Minimum exhaust velocity added to Ship velocity.
maxExhaustVelocity number Maximum exhaust velocity added to Ship velocity.
thrusterAcceleration number How much the Ship accelerates from thrusting.
thrusterParticles number How many Particles to spawn each iteration while thrusting.
shipBorderRebound number Multiplied to Ships' velocity when bouncing off borders.
shipBorderFriction number Multiplied to Ships' velocity when dragging along a border.
shipDrag number Multiplied to Ships' velocity each iteration to simulate the drag of an atmosphere.
particleDrag number Artificial higher drag applied to Particles.
particleType string This property is used to select a particle effect type with dat.GUI. This setting itself does not change the effect, but there is an onChange() method on the GUI to change it when this value is changed in the GUI.
nullifyThreshold number The threshold at which velocity will be nullified instead of reflected buring border collisions.
ships Array Where all Ships in the simulation are stored.
playerID number The index of which Ship in Game.ships is being controlled.
loop function Main game loop.
nullifyLowVelocity function Nullifies low velocities on objects passed to it.
