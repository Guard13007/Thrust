Class: Particle

Particle

new Particle(locationopt, effectopt, sizeopt, speedopt, parentopt)

Creates a rectangular object to be used with other Particles for basic particle effects.
Parameters:
Name Type Attributes Default Description
location object <optional>
 {x:0,y:0} Center of Particle.
effect object <optional>
 {r:255,g:255,b:255,method:function()} Define starting color and method to execute on each call to Particle.effect()
size object <optional>
 {width:1,height:1} Size of Particle.
speed object <optional>
 {x:0,y:0} Velocity of Particle.
parent object | Array <optional>
 A parent object can be specified, helps with the remove() method.
Properties:
Name Type Attributes Description
x number X position.
y number Y position.
color Array Color as [R,G,B]
effect function Called to modify Particle.
width number Width.
height number Height.
v vector Velocity.
parent object <optional>
 Parent object, used in Particle.remove()
onCreate function Called if it exists when it is created.
